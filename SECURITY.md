# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SSH Vault, please report it responsibly by emailing:

**github@seniorjava.dev**

Please do **not** open a public GitHub issue for security vulnerabilities.

## What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response

I'll acknowledge your report within 48 hours and aim to release a fix as soon as possible.
